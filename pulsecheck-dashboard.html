Embed the feedback widget

Add this script tag to your website. Customers submit feedback and results are analyzed in real-time and streamed to your PulseCheck dashboard.

<!-- PulseCheck Widget --> <script src="https://cdn.pulsecheck.io/widget.js" data-workspace="acme-corp-xxxx" data-position="bottom-right" data-theme="dark" ></script>

REST API

Send feedback programmatically from your backend. Works with any language or framework.

POST https://api.pulsecheck.io/v1/analyze { "text": "The product was great but shipping took forever", "model": "smart", "source": "checkout-survey" }

Webhook

Get notified instantly when a negative review is detected. POST to your endpoint with the full analysis payload.

{ "event": "negative_detected", "confidence": 0.91, "text": "Worst experience I've had...", "source": "product-page", "timestamp": "2025-04-30T09:41:00Z" }
